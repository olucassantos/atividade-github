1 - Inserindo um projeto novo no github.
    1.1 - Via VS Code;
        1.1.1 - Configurar o usuarío e email do git:
            git config --global user.name "seu nome"
            git config --global user.email "seu email"

        - Na aba Código Fonte do VSCode, existe o botão "Publicar no GitHub", que abre uma janela para inserir o nome do repositório e a descrição do projeto.
    1.2 - Via terminal:
        1.2.1 - Criar um repositório no github;
        1.2.2 - Clonar o repositório:
            git clone
        1.2.3 - Criar um arquivo README.md;
        1.2.4 - Adicionar o arquivo ao git:
            git add README.md
        1.2.5 - Fazer o commit do arquivo:
            git commit -m "Primeiro commit"
        1.2.6 - Enviar o arquivo para o github:
            git push -u origin master
